# 05 ─ 審查（三）：偽陽性與偽陰性，門檻的倫理

> 這一章是全站招牌互動「偽陽性分數線」的論述母本：一條分數線劃下去，誰被誤殺？

## 5.1 一條線，兩種代價

任何分數加門檻的系統，都在同一條線上同時決定兩件事：

| | 定義 | 代價 |
|---|---|---|
| **偽陽性（false positive）** | 正常內容被誤判為垃圾 | **誤傷正常使用者——言論自由的代價** |
| **偽陰性（false negative）** | 垃圾沒被抓到 | **漏接 spam——公共空間品質的代價** |

〔TWIGF〕門檻往下拉，偽陰性減少（垃圾抓更多），但偽陽性上升（誤殺更多）；往上抬則相反。**兩者不可能同時為零**，這是數學事實，不是工程疏失。Duke 的白話：

> 「既然只是一個能盡可能涵蓋 spam 文章的分數，那當然就會有一些偽陽性……正常的文章但被誤判，或者沒找出來的 spam（偽陰性），這兩件事一定沒辦法完全滿足。」〔TWIGF〕

## 5.2 Matters 版的「base rate」問題：誤殺率低，但被誤殺的價值密度極高

教科書上的偽陽性陷阱是「當垃圾很稀有時，即使 99% 準確的過濾器也會產生大量誤告」。Matters 的情況是它的**鏡像**，更尖銳：

- 平台日常內容約 **70–80% 是垃圾**，所以垃圾不稀有；模型的**絕對誤殺率很低**（第 03 章：3.1%→0.13%）。
- 但**受保護的公民、政治、調查內容只佔約 1%**——而模型「錯殺放大」時，剛好就是把政治評論、學術考據、流亡記者的敏感書寫誤判為垃圾（第 03 章「可樂倒進麵團」）。

換句話說：**即使誤判率低到 0.13%，被誤殺的那一小撮，剛好是平台存在理由的那 1%。** 這就是為什麼那條不對稱原則必須存在——

> 「在不確定時，寧可漏看一篇垃圾，不願消音一篇表達。」〔提案〕〔spam:D1〕

## 5.3 工程上的對策：conformal 棄權決策

新一代的處置不再「分數過線就硬判」，而是加上一層**conformal（保形）棄權決策**：不確定的不判，丟給人複核。〔repo〕〔spam:D2〕

- 輸出不再只是一個分數，而是 **`decision ∈ {allow, block, review}`**：明確的放行、封鎖，與「**我不確定，交給人**」。
- 它提供統計保證：在校準下，「真正合法內容被誤擋」的機率被控制在一個上界 ε 之內，且與類別先驗無關。
- 效果（文章模型）：真實**誤殺率從 24.0% 壓到 1.6%**（@0.5 門檻）；在 ε=0.02 下，合法內容誤擋約 **1.8–1.9%**、召回約 **93%**。〔repo〕〔spam:D2〕
- 每個處置附帶一個「理由字串」，對應歐盟 DSA 第 17 條的「理由說明」義務。〔repo〕

這層設計把「灰區」明確地交還給人，是「降低處置不可逆成本」這條主線（第 06 章）在偵測端的具體實作。

## 5.4 門檻不該由工程師獨斷

最關鍵的治理命題，不是技術問題，而是價值問題：**那條線該劃在哪裡？**

實務上，門檻是用「一個非常古老的辦法」定出來的——「反覆去測試怎麼樣的數字可以涵蓋最多的 spam 文章……以體驗為主」。〔TWIGF〕但簡報把它升格為倫理命題：

> 「這個權衡點不該由工程師獨自決定，這是工作坊的核心命題。」〔g0v／TWIGF 簡報〕

這正是本計畫舉辦閉門與公開研討、把「偽陽／偽陰的門檻往哪邊調」當成第一道罐頭問題的原因。〔g0v〕

## 5.5 藝術的邊界

偽陽性不只是統計問題，有時是定義問題。Duke 親自舉的難題：

> 「如果廣告文寫得非常值得閱讀，或非常令人過癮，那這樣到底算不算 spam？……我還記得就是有寫心思的、或者是寫詞的，那種非常前衛的藝術，可能不是工程可以理解的部分。」〔TWIGF〕

當「垃圾」與「前衛創作」在文本特徵上難以區分，任何門檻都會踩到灰色地帶——這是模型本質上無法獨力解決的。

## 5.6 實證：申訴極少，反而透露了什麼

一個反直覺的觀察：過去幾年，因處置而來的**正式申訴只有極少數**（內部討論中提到「只有一起」房屋仲介詢問為何被禁言、確認是真人外包公關後已解封）。〔小黑屋〕【脫敏】

這不代表系統完美，反而推論出：**偽陽性極低，但偽陰性比例很高**（沒抓到的真垃圾很多）。〔小黑屋〕運營成員也坦承曾誤刪真人：一位使用者寫了很多產品介紹文，被誤以為是業配而誤註銷，來信後才道歉復原。〔小黑屋〕【脫敏】

此外，公開的研究資料集也誠實揭露：早期有「作者被限制 → 其所有內容被標為垃圾」的代理標籤瑕疵，誤掃了中文政治／學術／創作長文；約 5,000 筆已人工修正，但未修正的中文長文仍可能殘餘 **5–15%** 誤標。〔repo〕〔spam:D2〕**把這件事講出來，本身就是透明開盒。**

## 小結

偽陽性是這個計畫的情感與倫理核心：它把抽象的「言論自由 vs 公共品質」變成一條看得見、可以拖動的分數線。網站的招牌互動，就是讓使用者親手拉動這條線，看著垃圾減少的同時，那 1% 該被保護的內容如何一起被掃進去。下一章，我們離開偵測，進入「抓到之後怎麼處置」——七種手段的光譜。
