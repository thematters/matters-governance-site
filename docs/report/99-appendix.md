## B. 術語表

| 術語 | 說明 |
|---|---|
| **IPFS**（InterPlanetary File System） | 分散式儲存協定。以「內容指紋（CID）」定址，只要有節點持有就讀得到，使網域／URL 封鎖失效。Matters 自 2018.10 起所有文章上鏈。 |
| **IPNS** | 為 IPFS 內容提供「固定地址 → 可更新指紋」的指針，像網址一樣永遠指向最新內容，可被 RSS 訂閱。 |
| **ENS** | 以太坊命名服務。用好記的 `.eth` 名稱指向 IPNS／內容地址。 |
| **CID（內容指紋）** | IPFS 對一份內容算出的唯一雜湊；內容一改，CID 就變。 |
| **DNS-RPZ**（Response Policy Zone） | DNS 層級的政策響應機制，可改寫解析結果以封鎖特定網域。2025 春節 Matters 即因此被誤封（第 10 章）。 |
| **小黑屋** | 把帳號／文章從首頁與公共頻道排除，但帳號仍 active、直連 URL 仍可開——可逆的可見度處分（第 06 章）。 |
| **降噪** | 依 spam 分數在排序 SQL 直接排除，自動化的可見度處分。 |
| **打掃** | 守望相助隊移除留言並公開存證的動作（removal）。 |
| **偽陽性 / 偽陰性** | 偽陽＝正常內容被誤判為垃圾（言論自由代價）；偽陰＝垃圾沒被抓到（公共品質代價）。 |
| **conformal（保形）決策** | 一層棄權機制：輸出 allow／block／**review**，不確定的交給人，並對「合法內容被誤擋」提供統計上界（第 05 章）。 |
| **Ring（集團）** | 同一模板被大量帳號協同重複張貼的行為樣態；靠繁簡折疊＋相似度聚群偵測，抓行為不抓字句（第 04 章）。 |
| **二次詐騙文** | 偽裝成「幫你追回被詐騙的錢」、實則進行第二輪詐騙的內容；DNS-RPZ 事件的引信。 |
| **LoRA** | 低秩微調，用少量參數微調大模型。Matters 文章模型＝granite embedding + LoRA。 |
| **DSA** | 歐盟《數位服務法》。其「移除／停權／可見度限制」三類處分、第 17 條「理由說明」義務，是本報告的法律對照框架。 |
| **CIB**（Coordinated Inauthentic Behavior） | 協同造假行為；政治網軍與集團洗版的上位概念。 |
| **守望相助隊 / 海巡 bot** | 群眾協力的人工與自動化兩條路線（第 08 章）。 |
